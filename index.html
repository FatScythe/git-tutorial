Git And Github

To open cmd on win7 windows+R input cmd

Open a folder with gitbash

To know the version $git --version
To create a new file in folder $touch index.html
To create a new repository $git init, To Add name and email to git
$git config --global user.name 'Your Name here'
$git config --global user.email 'Your email here'
To add specific file to the local repository
$git add index.html
To check status of added files
$git status To remove files from the local repository
$git rm --cached index.html
To add all of the files of a specicific file type(e.g html)
git add *.html
To add all the files and regardless of file type or extension
$git add .
To commit files (1st Method)
$git commit
Press I>>NB:# means its commented, So remove the # to name the commit>>(to get out)>>Click esc>>type:wq>> Press Enter
To commit files to the master branch (2nd Method)
$git commit -m 'Comment here'
